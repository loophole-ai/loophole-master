# Loophole Master

The central repository for [Loophole IDE](https://github.com/loophole-ai/loophole-ide) — an AI-powered code editor built by Garv Agnihotri.

## Repositories

| Repo | Description |
|------|-------------|
| [loophole-ide](https://github.com/loophole-ai/loophole-ide) | The main IDE — VSCode-based editor with AI agent core |
| [versions](https://github.com/loophole-ai/versions) | Update feed — serves latest.json for auto-updates |
| [loophole-cli](https://github.com/loophole-ai/loophole-cli) | CLI tool for Loophole |
| [.loopholerules](https://github.com/loophole-ai/.loopholerules) | Default AI rules and context for Loophole projects |

## Getting Started

Clone everything at once:

```bash
git clone --recurse-submodules https://github.com/loophole-ai/loophole-master
```

Or if you already cloned:

```bash
bash setup.sh
```

## Updating All Repos

```bash
bash update.sh
```

Each repo has its own issues and PRs. Visit the individual repo linked above to contribute.
